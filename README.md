# Evan Petersen - ML Engineer Portfolio

My personal portfolio website featuring an immersive space-themed design. Built to showcase my machine learning projects, certifications, and software development experience with interactive animations and modern web technologies.

## About This Project

This portfolio features:
- **Space-themed design**: Animated starry backgrounds transitioning from deep space to sun atmosphere with custom CSS animations
- **Interactive project showcase**: Machine learning and AI projects with live demos, videos, and detailed descriptions
- **Animated education timeline**: Rocket journey animation along SVG path with satellite data panels
- **Floating certification satellites**: Professional certifications displayed as articulated satellites with solar panels and status LEDs
- **Working contact form**: EmailJS integration with real-time messaging on an animated sun surface background
- **Performance optimized**: React useMemo hooks preventing animation jitters, smooth Framer Motion transitions

## Technical Highlights

- React 18 with functional components and hooks
- Vite for development and optimized production builds  
- Tailwind CSS utility-first styling with custom gradient classes
- Framer Motion for physics-based animations and gestures
- EmailJS for serverless contact form functionality
- Performance optimization with useMemo to stabilize animation positions
